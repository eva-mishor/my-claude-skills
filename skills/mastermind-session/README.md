# Mastermind Session Skill

Interactive mastermind sessions with 5-10 thought leaders to solve specific business, strategic, or personal challenges.

## Installation

1. Download and extract this ZIP file
2. In Claude.ai, go to your account settings
3. Navigate to the "Skills" section
4. Click "Upload Skill"
5. Select the entire `mastermind-session-skill` folder

## Usage

**Activate the skill by saying:**
- "Mastermind session"
- "Let's start a mastermind"
- "I need expert perspectives on..."
- "Can we convene a mastermind about..."

## What This Skill Does

The Mastermind Session skill creates an interactive discussion with carefully selected thought leaders who:

✓ Engage in dynamic dialogue based on your specific challenge  
✓ Challenge assumptions and reveal blind spots  
✓ Disagree productively to illuminate different angles  
✓ Push toward actionable decisions  
✓ Reference your past conversations for context  

## Features

- **Context-Aware**: Automatically searches past mastermind sessions and recent conversations
- **Customizable Panel**: Choose from 40+ thought leaders across different domains
- **Interactive**: You're an active participant, not a passive observer
- **Flexible**: Summon additional experts mid-discussion
- **Action-Oriented**: Drives toward clear decisions and next steps

## Example Session

```
You: "Mastermind session. I'm stuck on whether to focus on my Swedish customer 
service directory or build a new AI consulting offer."

Claude: [searches past conversations and recent context]

Based on this challenge, I recommend:

1. Cal Newport - Deep work, focus, essentialism
2. Dan Sullivan - Strategic thinking, Who Not How
3. Seth Godin - Marketing, positioning, tribes
4. Tim Ferriss - 80/20, efficiency, testing
5. Gay Hendricks - Zone of genius, upper limits

Who would you like to include in today's session?

You: "Let's go with Cal, Dan, Seth, and Gay."

Claude: Perfect. Let's begin...

[Interactive discussion unfolds with you responding to each expert]
```

## Session Structure

1. **Goal Setting** - Define the specific challenge
2. **Expert Selection** - Choose 5-10 relevant thought leaders  
3. **Context Import** - Profiles are loaded for each expert
4. **Interactive Discussion** - Dynamic conversation with you participating
5. **Wrap-Up** - Clear summary and action items (when you request it)

## Tips for Best Results

- **Be specific** about your challenge or decision
- **Engage actively** - respond to expert questions
- **Take your time** - let the discussion develop naturally
- **Redirect freely** - ask specific experts to weigh in
- **Request wrap-up** when you're ready to conclude
